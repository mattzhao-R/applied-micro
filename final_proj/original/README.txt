Replication files for:

Jawad M. Addoum
"Household Portfolio Choice and Retirement"
Review of Economics and Statistics

This replication kit includes:
Stata do file (HHPortChoiceRetirement_Code)
Four Stata 14 dta files (cpi_base2012, hd2sp_laborinc_ratio, HRS_sample_final, and risk_aversion)

To replicate the results in the paper, put all files in one folder, change the Stata working directory to that folder, and run the do file.
